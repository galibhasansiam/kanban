About This App

Kanban Task Manager is a simple task organizer that helps you track work using three columns: To Do, In Progress, and Done. It is built entirely with HTML, CSS, and JavaScript, and it saves your tasks in the browser using localStorage, so your board stays the same even after you close or refresh the page.

How to Use

  1. Type a task in the input box at the top of the board.
  2. Choose which column the task belongs to (To Do, In Progress, or Done).
  3. Click "Add Task" to place it on the board.
  4. Drag and drop a task card into a different column to change its status.
  5. Click "Edit" on a card to change its text, or "Delete" to remove it.

Features

  • Add, edit, and delete tasks
  • Move tasks between columns with drag and drop
  • Task counters for each column
  • Data is saved automatically in the browser (localStorage)
  • Responsive layout that works on smaller screens

Built With

HTML5, CSS3, and vanilla JavaScript (no frameworks or libraries).
