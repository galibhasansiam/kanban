# Kanban Task Manager

A simple Kanban-style task manager built with plain HTML, CSS, and JavaScript.
Made for the CSE 479 (Web Programming) assignment.

## Live Demo
https://galibhasansiam.github.io/kanban/

## Features
- Add, edit, and delete tasks
- Three columns: To Do, In Progress, Done
- Drag and drop tasks between columns
- Task counters per column
- Tasks are saved in the browser with localStorage
- Responsive layout
- About page explaining how to use the app

## Files
- `index.html` - main board page
- `about.html` - info page about the app
- `style.css` - styling
- `script.js` - app logic

## Tech
HTML5, CSS3, vanilla JavaScript. No frameworks or libraries.
